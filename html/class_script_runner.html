Detailed Description

Uruchamia skrypty Python i raportuje wyniki przez sygnały Qt.

Używa QProcess do uruchomienia skryptu bez blokowania UI. Wyniki (stdout, stderr, kod wyjścia) są przekazywane przez sygnały.
